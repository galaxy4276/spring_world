## 프로젝트 목적
* Spring Boot 프레임워크에 대한 이해도 및 실력 향상
* JPA 에 대한 이해도 및 실력 향상, SQL 작성 능력 향상
* 캐시 DB 에 대한 이해도 및 실력 향상
* React.js + React Query + tailWindCSS 를 이용한 개발 경험 ( 개발 트렌드 잡기 )
* 웹 페이지 반응형 개발 학습
* 리액트를 사용하는 개발 실력 향상
* 웹에 대한 심화적인 이해
* 웹 소켓에 대해 이해도 및 실력 향상
